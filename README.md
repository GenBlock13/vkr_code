# Детектор аномальных отзывов маркетплейса — альфа-версия

Система классификации русскоязычных отзывов на подлинные и имеющие признаки
автоматической генерации (TF-IDF char n-grams + Random Forest).
ВКР, ТГУ им. Г.Р. Державина, Еременко Г.С.

## Быстрый старт

```bash
pip install -r requirements.txt
python bot_detector.py        # обучение модели → model_pipeline.pkl, results.json
streamlit run app.py          # запуск веб-интерфейса (http://localhost:8501)
```

## Состав

| Файл | Назначение |
|---|---|
| `app.py` | Streamlit-интерфейс: дашборд + 3 рабочие вкладки |
| `bot_detector.py` | Обучение модели (синхронизированные гиперпараметры) |
| `utils.py` | extract_features(), генератор датасета 400+400 |
| `marketplace_loader.py` | WB-ссылка, поиск по названию, generic-парсер, демо-режим |
| `demo_data/*.json` | 3 офлайн-снимка отзывов для демонстрации |
| `model_pipeline.pkl` | Обученный Pipeline (joblib) |
| `results.json` | Метрики последнего обучения |

## Функции альфа-версии

- **Дашборд**: Accuracy / ROC-AUC / OOB / CV, матрица ошибок, топ-15 n-грамм (Plotly).
- **Проверка отзыва**: одиночная классификация + диагностика 6 признаков.
- **Файлы**: загрузка TXT / CSV / JSON, пакетная классификация, сохранение
  результатов в CSV / JSON.
- **Маркетплейс**: отзывы по ссылке Wildberries; поиск товара по названию
  (например, «Средство от комаров детское»); generic-парсер произвольной
  страницы (BeautifulSoup); демо-режим — работает полностью офлайн.

## Метрики текущей модели (запуск 2026-06-12)

Accuracy 98,75 % · ROC-AUC 1,0000 · OOB 0,9953 · CV F1(macro) 0,9925 ± 0,0025
Матрица ошибок: TN=80, FP=0, FN=2, TP=78.

## Ограничения и этика

- Сетевые запросы: тайм-аут 10 с, пауза ≥ 1 с (НФТ-N3), без обхода защиты.
- Имена авторов отзывов не сохраняются.
- Метки модели — оценка по текстовым признакам, не установленный факт накрутки.
- Generic-парсер не работает на SPA-страницах (JS-рендеринг) — для них
  предусмотрены загрузка из файла и демо-режим.
