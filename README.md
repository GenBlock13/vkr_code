# ReviewGuard — Детектор аномальных отзывов маркетплейса

Система классификации русскоязычных отзывов на **подлинные** и **имеющие признаки
автоматической генерации** на основе TF-IDF char n-grams + Random Forest.

## Состав репозитория

| Файл / папка | Назначение |

| `launcher.py` | GUI-лаунчер (Tkinter): обучение и запуск веб-интерфейса из одного окна |

## Запуск из исходного кода (режим разработки)

### Требования

- **Python 3.10 или новее** (проверить: `python --version`)
- Windows 10 / 11 (для других ОС — аналогично через bash)
- Доступ в интернет для установки зависимостей

### Шаги

```bat
:: 1. Клонировать репозиторий
git clone https://github.com/<ваш-логин>/review_guard.git
cd review_guard

:: 2. Создать и активировать виртуальное окружение
python -m venv .venv
.venv\Scripts\activate

:: 3. Установить зависимости
pip install -r requirements.txt

:: 4. Запустить GUI-лаунчер
python launcher.py
```

Лаунчер предложит обучить модель при первом запуске.
После обучения нажмите «Открыть веб-интерфейс» — браузер откроется автоматически.

**Альтернативно** (без лаунчера, только командная строка):

```bat
python bot_detector.py        :: обучение → model_pipeline.pkl, results.json
streamlit run app.py          :: веб-интерфейс на http://localhost:8501
```

## Сборка портативного EXE (Windows)

EXE-файл позволяет запускать программу на любом Windows-ПК **без установки Python**.

### Требования для сборки

- Python 3.10+ с доступом в интернет
- ~500 МБ свободного места на диске (зависимости + сборка)

### Шаг 1 — создать окружение и установить зависимости

```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### Шаг 2 — запустить скрипт сборки

```bat
build.bat
```

Скрипт автоматически:
1. Установит PyInstaller
2. Соберёт `dist\ReviewGuard\` со всеми зависимостями (~350–400 МБ)
3. Выведет итоговый путь к EXE

> ⏱ Сборка занимает **2–5 минут** в зависимости от скорости диска.

### Шаг 3 — готово

```
dist\
└── ReviewGuard\
    ├── ReviewGuard.exe   ← запускать этот файл
    └── _internal\        ← не удалять, нужна для работы EXE
```

**Для переноса на флешку** скопируйте папку `dist\ReviewGuard\` целиком.
На целевом ПК запускайте `ReviewGuard.exe` — Python не требуется.

> ⚠️ Не запускайте EXE из папки `build\` — это промежуточный артефакт без нужных библиотек.

## Ограничения и этика

- Сетевые запросы: тайм-аут 10 с, пауза ≥ 1 с (НФТ-N3), без обхода защиты.
- Имена авторов отзывов не сохраняются.
- Метки модели — оценка по текстовым признакам, а не установленный факт накрутки.
- HTML-парсер не работает на SPA-страницах (JS-рендеринг) — используйте файл или демо-режим.
- Wildberries API работает без авторизации; структура ответов может измениться.
